Simple app that lists top 100 GitHub users that are living in Thailand, ordered by follower count.

Note that there's an API limit which will allow only 5 call per minutes. So if you see blank page, simply hit refresh until something shows up.
About Rate limit.

We can get more info for users such as real name, follower count but that require me to register this app as an OAuth app on GitHub which I'm too lazy to do.
